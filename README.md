# WGUPS Routing Program
A Python program that simulates package delivery for a logistics company. This is a school project that demonstrates truck routing optimization, package tracking, and delivery scheduling using hash tables and greedy algorithms.

## Features
- Optimized Truck Routing – Calculates efficient routes for multiple trucks to minimize mileage
- Package Delivery Tracking – Track the delivery status of individual packages at any given time
- Load Packages by Address – Packages are assigned to trucks based on their delivery addresses
- Time-Based Status Reports – View package status at any time of the day
- CSV Input & Output – Load addresses, distances, and package information from CSV files

## Installation
1 - Clone the repo:
`git clone https://github.com/yourusername/wgups-routing.git`

2 - Go into directory:
`cd wgups-routing`

3 - Make sure you have Python 3 installed

4 - Run
`python main.py`

5 - Follow the on-screen menu

## Usage
(1) Quit - Exits the program

(2) Search Package Status - Enter a package ID and a time (HH:MM) to see its status

(3) View All Packages Report - Enter a time (HH:MM) to see the status of all packages
